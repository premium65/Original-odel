## Technical Notes

### Data Structure:
Each ad contains:
```typescript
{
  id: number,              // Unique identifier (1-27)
  title: string,           // Product name
  description: string,     // Product details
  imageUrl: string,        // Image URL
  targetUrl: string,       // Click destination
  price: string,           // Display price (decimal)
  reward: string,          // User earnings (decimal)
  type: "click",          // Ad type
  duration: 30,           // Seconds
  isActive: true,         // Visibility status
  totalViews: 0,          // View counter
  createdAt: Date         // Timestamp
}
```

### Storage:
- **In-Memory**: Available immediately in `inMemoryAds` array
- **Database**: Can be seeded using `npm run seed:ads`
- **Persistent**: Database ads survive server restarts
- **Fallback**: In-memory used when database unavailable
