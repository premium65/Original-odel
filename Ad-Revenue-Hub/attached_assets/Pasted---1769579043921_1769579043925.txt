OPTION 2: ADD VALUE
================================================================================

PURPOSE:
--------
Add amount to any user field. Can add POSITIVE or NEGATIVE values.

FIELDS YOU CAN ADD TO:
----------------------
- milestoneAmount
- ongoingMilestone
- milestoneReward
- destinationAmount
- totalAdsCompleted
- points

HOW TO USE:
-----------
1. Select user
2. Click "Add Value" tab
3. Choose field from dropdown
4. Enter amount to add (can be negative like -5000)
5. Click "Add"

CALCULATION:
------------
New Value = Current Value + Added Amount

EXAMPLES:
---------
Example 1: Add positive value
  Before: milestoneAmount = 1,000 LKR
  Action: Add 500 to milestoneAmount
  After: milestoneAmount = 1,500 LKR

Example 2: Add negative value (to set negative balance)
  Before: milestoneAmount = 0 LKR
  Action: Add -5000 to milestoneAmount
  After: milestoneAmount = -5,000 LKR

                    PART 14: CALCULATION FORMULAS
================================================================================
================================================================================


NEW USER REGISTRATION:
----------------------
milestoneAmount = 25000 (bonus)
status = "pending"
totalAdsCompleted = 0


FIRST AD CLICK (Bonus User):
----------------------------
IF milestoneAmount == 25000 AND totalAdsCompleted == 0:
    milestoneAmount = 0
    totalAdsCompleted = 1


NORMAL AD CLICK:
----------------
milestoneAmount = milestoneAmount + commission

Example:
  Before: -5000
  Commission: 100
  After: -5000 + 100 = -4900


PAYOUT (Approved):
------------------
milestoneAmount = milestoneAmount - payoutAmount

Example:
  Before: 5000
  Payout: 3000
  After: 5000 - 3000 = 2000


DEPOSIT:
--------
milestoneAmount = milestoneAmount + depositAmount


RESET FIELD:
------------
field = 0


ADD VALUE:
----------
field = field + amount


PAYOUT BUTTON VISIBLE:
----------------------
IF totalAdsCompleted >= 28:
    SHOW button
ELSE:
    HIDE button


DEPOSIT BLOCKING:
-----------------
IF milestoneAmount < 0 AND hasDeposit == false:
    BLOCK ad clicking
ELSE:
    ALLOW ad clicking


DAILY RESET (Midnight):
-----------------------
milestoneReward = 0
(All other fields unchanged)
