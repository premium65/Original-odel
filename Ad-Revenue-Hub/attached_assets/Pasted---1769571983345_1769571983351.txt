2. Clicks on available ad

5. Balance updates in REAL-TIME after each click

INSTANT UPDATE RULE:

Every ad click â†’ milestoneAmount updates IMMEDIATELY (not end of day)

CALCULATION:

PART 4: PAYOUT SYSTEM
================================================================================

PAYOUT BUTTON VISIBILITY:
-------------------------
- Payout button is HIDDEN until user completes 28 ads
- After 28 ads completed â†’ Payout button APPEARS

Rule:
  IF totalAdsCompleted >= 28 â†’ Show Payout Button
  IF totalAdsCompleted < 28 â†’ Hide Payout Button

Example:
  User completed 15 ads â†’ Payout button HIDDEN
  User completed 28 ads â†’ Payout button VISIBLE âœ…
  User completed 50 ads â†’ Payout button VISIBLE âœ…

PAYOUT PROCESS:
---------------
1. User clicks Payout button (only visible after 28 ads)
2. User enters amount to payout
3. User enters bank details
4. Submits payout request
5. Status = "pending"
6. Admin sees in Admin Panel â†’ Payouts
7. Admin approves or rejects

ADMIN APPROVES:
---------------
- User's milestoneAmount DECREASES by payout amount
- Payout status = "approved"
- Admin pays user manually (bank transfer)

ADMIN REJECTS:
--------------
- User's milestoneAmount stays same
- Payout status = "rejected"
- User sees rejection reason

PAYOUT CALCULATION:
-------------------
  New Balance = Current Balance - Payout Amount

Example:
  Balance = 5000 LKR
  Payout request = 3000 LKR
  Admin approves
  New Balance = 5000 - 3000 = 2000 LKR

PART 6: PROMOTION/RESTRICTION SYSTEM
================================================================================

WHAT IS PROMOTION:
------------------
Special earning rules set by admin for specific user

ADMIN SETS:
-----------
- Ads Limit: Number of ads user must complete (e.g., 100)
- Deposit: Display amount (e.g., 5000)
- Commission: Earning per ad (e.g., 50 LKR)

HOW IT WORKS:
-------------
1. Admin sets promotion on user
2. User must click X number of ads
3. Each ad = promotion commission (not normal ad price)
4. After completing all ads â†’ promotion ends
5. User returns to normal mode

Example:
  Promotion: 100 ads, 50 LKR commission
  User clicks 100 ads
  Total earned = 100 Ã— 50 = 5000 LKR
  After 100 ads â†’ promotion ends
